rumcap capture demo

This page uses the rumcap streaming Encoder to capture its own Web Performance data (navigation, resources, paint/LCP, layout shifts, interactions, long tasks/LoAF, user timing, the JS self-profiler) plus a custom demo-app timeline, and packs it into a .rcap file.

Interact with the page (the button below), then click Save capture — or just switch tabs; it auto-saves on hide. Open the downloaded .rcap in a supporting viewer (waterfall-tools).
